## 一、最毒禁用句式（出现即修，最高优先级）

| 毒级 | 句式 | 错误例 | 修法 |
|------|------|--------|------|
| ★★★★★ | "不是A，而是B" / "不是A，不是B，而是C" | "他不是冷漠，而是绝望" | 直接写 B 或用更自然的表达 |
| ★★★★ | "，带着……" 万能状语 | "他笑了一下，带着一丝不易察觉的嘲讽" | 拆短句或换动作 |
| ★★★★ | "声音不大，却带着一种……的力量" | "她声音不大，却带着不容置疑的力量" | 直接写声音特征或动作 |
| ★★★★ | "他/她知道……" | "他知道这一切都来不及了" | 用行为展示认知 |
| ★★★ | "仿佛/犹如/宛若……一般" | "仿佛能穿透一切一般" | 删掉或白描 |
| ★★★ | "眼中闪过一丝……" / "嘴角勾起一抹……" | "眼中闪过一丝悲伤" | 他垂下眼 / 他笑了一下，没到眼底 |
| ★★★ | "心中涌起一股……" / "心头一震" | "心中涌起一股暖流" | 用身体反应 |
| ★★ | 章末预告 "他不知道的是……" | "他不知道的是，更大的风暴即将来临" | 用具体钩子物件/事件收束，避免空泛预告 |

## 二、一级禁用词（出现即替换）

### 情态类
仿佛、犹如、宛若、如同、一丝、一抹、些许、几分、隐约

### 动作类

### 心理类
心中一动、心头一震、心下了然、心中暗道、心底泛起、不由得

### 判断类
不容置疑、不容置喙、不易察觉、显而易见、毫无疑问、不可否认

### 形容类
坚定、闪烁着光芒、狡黠、深邃、凛冽、冰冷

### 过渡类
不由自主、情不自禁、自然而然

## 三、二级禁用词（高频出现时替换）

突然、好像、瞬间
（角色口语、真实突发、时间压缩、视角不确定时可保留）

|--------|-----------|
| 瓦解 | 消失 / 散了 / 没了 |
| 无名火 | 烦躁 |
| 往我心上捅刀子 | 心烦意乱 |

- "他/她终于明白..."
- "他/她这才意识到..."
- "此刻，他/她..."

- 连续3句以上相同结构的排比
- "有的...有的...有的..."
- "一边...一边...一边..."

### 升华句式
- "这一刻..."
- "他知道..."

## 四、中文 AI 高频连接词/填充词

### 连接词（过度使用时标记）
此外、同时、值得注意的是、需要指出的是、不难发现、基于以上分析、综上所述、总而言之、首先/其次/最后、一方面/另一方面、不可否认、毋庸置疑、随着...的发展、在...的背景下、从某种意义上说、在当今社会、至关重要、深入探讨、进一步来说、换言之、由此可见

### 填充短语（直接删除）
- "值得一提的是"
- "从某种意义上说"
- "不可否认的是"
- "毋庸置疑"
- "事实上"
- "不得不说"
- "坦率地说"
- "客观来讲"
- "众所周知"
- "毫无疑问"

### 虚假宣告（只宣布重要性，未指明具体内容）
- "意义重大"
- "影响深远"
- "问题的根源在于结构性因素"
- "其重要性不言而喻"
- "这引发了广泛的讨论"

## 五、比喻分类（默认全删）

| 比喻类别 | 例 |
|---------|----|
| 动物类 | "像一头被抛弃的野狗" |
| 物品/现象类 | "像一把刀" "脸色惨白得像这漫天的雪" |
| 状态类（陈词滥调） | "梨花带雨" "如沐春风" |
| 抽象类 | "像命运的齿轮" "像上辈子的尘埃" |
| 假设类 | "力道大得像是要把骨头捏碎" |

**处理原则**：把比喻改为直接描述／动词／名词／作用／结果／事实。例："脸色惨白得像这漫天的雪" → "脸色惨白"。

## 六、替换策略速查

| 原文类型 | 替换方法 | 示例 |
|----------|----------|------|
| 抽象情绪词 | 具体动作 | "紧张" → "手在抖" |
| "感到XX" | 外化表现 | "感到愤怒" → "攥紧拳头" |
| 形容词堆砌 | 白描手法 | "美丽动人的笑容" → "她笑了" |
| 书面表达 | 口语化 | "不容置疑" → "就是" |
| 解释性描写 | 留白 | "他因为害怕而..." → "他退后一步" |
| 连续排比 | 保留最强一条 | 3 句排比留 1 句 |
| 总结升华句 | 直接删除 | "这一刻，她终于明白了..." → 删 |
| "不是A，而是B" | 直接写 B | "他不是冷漠，而是绝望" → 直接写 B |
| 多余修饰 | 删 | "白色的药片" → "药片"；"手里那截链子" → "链子" |

## 八、高频踩雷词（来自 khazix-writer，出现即修）

这些词/句式一出现立刻暴露AI味，必须替换：

| 禁用表达 | 问题 | 替换建议 |
|----------|------|----------|
| "说白了" | AI特别爱用，一出现立刻暴露 | 删掉，直接说 |
| "意味着什么？" | AI标志性句式 | 删掉，或换成具体追问 |
| "这意味着" | 同上 | 换成更口语表达，如"也就是说"、"说到底就是" |
| "本质上" | 太学术 | 删掉，或用"说到底" |
| "换句话说" | 太书面 | 删掉，直接说 |
| "不可否认" | 套话 | 删掉 |
| "综上所述" / "总的来说" | 套话 | 删掉，用动作/场景收尾 |
| "首先...其次...最后" | 三段式 | 打散，用隐性连接 |
| "值得注意的是" / "不难发现" | 套话 | 删掉，直接说内容 |
| "让我们来看看" / "接下来让我们" | 套话 | 删掉，直接进入内容 |
| "在当今...的时代" | 教科书开头 | 换成具体事件切入 |
| "随着技术的不断进步" | 空话开头 | 换成具体事件切入 |
| "比如有一次..." | 假设性例子 | 换成"就像我今天正在搞的..."真实场景 |
| "某个AI工具" / "某个模型" | 空泛工具名 | 换成具体名称（Claude Code、DeepSeek等） |

## 十、推荐口语化词组（自带"活人感"）

> 来源：khazix-writer

### 转场和过渡
坦率的讲、说真的、我是真的觉得、反正我觉得、怎么说呢、其实吧、你想想看、我跟你说、回到xxx这块、这块需要注意一下、顺着上面的再聊聊

### 表达判断
我有时候觉得、我一直觉得、这话听着有点刺耳但、不是说xxx不行而是说、我自己的感受是、我始终坚信、我觉得还是挺重要的

### 承认和自嘲
说实话我也不确定、我自己也还在摸索、可能有些想法还不成熟、这个事儿我也踩过坑、愚钝如我、我说"理论上"是因为我自己还没完全跑通、说实话我们还差得远

### 情绪表达
这种感觉太爽了、我当时就愣住了、想想就觉得兴奋、我真的被震撼到了、搞得我现在还有点懵、太离谱了、给我一下子整不会了、这一下给我更干懵了、一时间无语凝噎、你敢信？？？

### 拉近距离
很多朋友可能不知道、可能有小伙伴纳闷、你如果关注这个领域的话、大家也都知道、看到的兄弟可以把公屏打在弹幕上

### 口头禅和口癖
这玩意、不是哥们、我寻思了一下我没寻思明白、有个屁的xxx、真的就是一声叹息、太牛逼了、比较骚的事
