# 🔥 10万+爆款选题炼金炉 · 6+ 内容形态矩阵

> 这是炼金炉的「分流注模」——一个好选题应该能裂变成多种内容形态，让不同创作者都能拿来用。
> 本文件定义 6 大主流形态的特性、节奏、爆款锚点、适配玩法。

---

## 形态总览（6 大类）

| 形态 | 核心载体 | 阅读时长 | 爆款锚点 | 适合创作者 |
|---|---|---|---|---|
| **① 文章标题** | 一行字 | 2 秒 | 钩子+利益承诺 | 所有平台、SEO、社群分发 |
| **② 文章角度** | 切入视角 | 30 秒决定 | 反共识/独家视角 | 公众号、知乎、专栏作者 |
| **③ 公众号长文** | 1500-3000 字 | 3-5 分钟 | 金句/数据图/故事 | 公众号编辑、内容运营 |
| **④ 短视频脚本** | 60-180 秒 | 完播率为王 | 前 3 秒钩子+反转 | 抖音/视频号/小红书博主 |
| **⑤ 漫画/插画** | 4-12 格 | 30-60 秒 | 表情/反差/梗图 | 漫画作者、插画师 |
| **⑥ H5/小游戏** | 互动 1-3 分钟 | 体验沉浸 | 结果页/人格证/海报 | 策划 PM、设计师 |

> 还有**⑦ 朋友圈短文+海报、⑧ 小红书图文、⑨ 信息长图、⑩ 直播话题**等附加形态，按需扩展。

---

## ① 文章标题 · 速成公式

**核心**：12 字内一锤定音。承接 12 套标题公式（详见 title-formulas.md）。
**适用平台**：公众号、知乎、头条、网易号、SEO 落地页。
**爆款关键**：
- 数字+利益（"7 个细节藏在……"）
- 反常识断言（"原来 X 不是 Y"）
- 身份对立（"不是 XX 看不懂"）
- 自问自答（"为什么 X？因为 Y"）

**字数限制**：朋友圈 ≤ 22 / 小红书 ≤ 20 / 公众号 ≤ 28 / 视频号 ≤ 22 / H5 海报主标 ≤ 14。

---

## ② 文章角度 · 视角设计

**核心**：同一个素材，不同切入角度产出完全不同的爆款。
**爆款角度模板**：
1. **个人经历型**：「我做了 X 年 Y 才发现……」
2. **反共识型**：「大家都说 A，但其实是 B」
3. **数据拆解型**：「XX 这件事我用 5 个数字讲清楚」
4. **历史纵深型**：「从 N 年前到今天，X 经历了什么」
5. **冷知识型**：「90% 人不知道，X 真正的运作方式」
6. **亲历者视角**：「作为 XX，我亲历了这件事」
7. **对比反差型**：「A 和 B，差在哪里」
8. **追问型**：「为什么没人问 X 这件事？」

**输出格式**：每个选题给 3-5 个角度选项，让创作者挑最契合自己人设的。

---

## ③ 公众号长文方案（1500-3000 字）

**结构骨架**：
1. **钩子段（200 字内）**：用故事 / 反差 / 数据 / 金句开场，制造阅读理由
2. **冲突段（400-600 字）**：把核心矛盾抛出来，让读者"想知道答案"
3. **论证段（800-1500 字）**：分 3-5 个小标题展开（信息源、案例、数据、对比）
4. **金句段（200 字内）**：1 句可截图传播的金句海报点
5. **行动段（200 字内）**：给读者一个动作（思考题/小工具/CTA）

**爆款细节**：
- **小标题数字化**："01 / 02 / 03"或"第一个真相 / 第二个真相"
- **段间金句加粗**：方便截图传播（**截图墙=朋友圈广告位**）
- **首图 + 文中配图 ≥ 3 张**：减少阅读疲劳
- **末尾留钩子**：互动话题或下篇预告

**输出格式**：给目录提纲（每个小标题 + 一句话内容简介）+ 1-2 段范文（钩子段必给）+ 1 张可截图金句海报文案。

## ④ 短视频脚本（60-180 秒）

**核心节奏（黄金 3-5-30 法则）**：
- **0-3 秒**：钩子！必须让用户停下来——视觉冲击/悬念金句/反差画面
- **4-15 秒**：抛出核心问题，"接下来我告诉你 XX"
- **15-90 秒**：3 个分点展开，每点配画面/字幕/对话
- **90-120 秒**：金句收束 + 转发钩子（"评论区聊聊""你也试试")

**视频形态选择**：
- **真人口播 + 大字幕**：知识/观点类
- **手绘 / 动效解说**：科普/历史/数据类
- **剧情演绎**：故事/吐槽/对话类
- **混剪 + BGM**：怀旧/情绪/氛围类

**输出格式**：
```
镜次 | 画面（含运镜/特效） | 文案/对白 | 音效/字幕 | 时长
1   | 全屏黑底+金句弹出  | "8 年没看球的我……" | 心跳音效 | 3s
2   | ……                | ……                  | ……      | 5s
```

**抖音/视频号字幕规则**：每条字幕 ≤ 12 字，每行停留 ≥ 1.5 秒。

## ⑤ 漫画 / 插画方案

**爆款漫画结构**：
- **4 格**：起 / 承 / 转 / 合（最适合朋友圈）
- **6-9 格**：长条形，适合公众号头图+小红书
- **12 格+**：长条故事，适合微博/B 站漫画号

**爆款基因**：
1. **反差表情包**：人物前后表情对比形成梗
2. **替代符号**：用物件代替抽象概念（"焦虑"画成乌云）
3. **金句画框**：把一句话放进画里当核心传播点
4. **彩蛋细节**：第二遍看才能发现的小细节，提升复看率

**输出格式**：每格写明
```
格1：[场景描述]｜人物：A（表情：XX）｜对白："……"｜彩蛋：左下角的 XX
格2：……
```

## ⑥ H5 / 小游戏方案

**18 类玩法快速选**（用户自有玩法库）：
- 测试型（MBTI 化）/ 选择型 / 拼图型 / 刷展型 / 养成型 / 地图型 / 时间线型
- 排行榜 / 转盘 / 合成 / 分支剧情 / 卡牌收集 / 抽取 / 对决 / 巡游 / 答题闯关 / 投票 / 跑酷

**核心要素**：
- **首屏 5 秒**：视觉锚点 + 1 句话玩法说明 + 1 个动效引导
- **3 步主流程**：① 用户输入/选择 → ② 系统反馈/演算 → ③ 结果页+分享
- **截图锚点页**：必有一张"用户截图发朋友圈"的结果海报（人格证/账单/勋章/时间轴）
- **声音默认静音**：右上角开关
- **版权规避**：自制插画/AI 生成视觉，避开官方 Logo/明星肖像/转播画面

**输出格式**：玩法类型 + 3 步主流程 + 首屏文案 + 结果页核心元素 + 视觉风格关键词。

## 多形态联动建议

一个核弹级选题，建议至少铺 **3 种以上形态**，吃透热度长尾：

| 时间窗口 | 形态推荐 | 目标 |
|---|---|---|
| 热点爆发 0-2 小时 | 朋友圈短文 + 海报 | 第一波传播抢首发 |
| 热点 2-12 小时 | 短视频/直播 | 流量黄金时段触达大众 |
| 第二天上午 | 公众号长文 | 深度+权威，承接收藏 |
| 第二天下午 | H5/小游戏 | 互动裂变，吃二轮流量 |
| 周末 | 漫画/合辑 | 怀旧/情怀沉淀，长尾分享 |

## 不同创作者人设速查

| 创作者类型 | 优先形态 | 心法重点 |
|---|---|---|
| 公众号编辑 | ③ 长文 + ① 标题 | 角度独家+金句海报 |
| 短视频博主 | ④ 短视频 + ① 标题 | 前 3 秒钩子+人设标签 |
| 漫画作者 | ⑤ 漫画 + ② 角度 | 表情反差+梗图化 |
| H5 策划 PM | ⑥ H5 + ② 角度 | 玩法落地+截图锚点 |
| 小红书博主 | ① 标题 + ⑤ 插画 | emoji+分隔符+人设 |
| 全栈运营 | 多形态联动 | 24h 内 3 种形态铺出 |
